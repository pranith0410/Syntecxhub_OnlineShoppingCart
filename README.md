# Online Shopping Cart (Java)

This project is developed as part of the **Syntecxhub Internship Program**.

## Project Description
A simple console-based Online Shopping Cart system built using Java Collections.

## Features
- Add items to cart
- View cart
- Calculate total price
- Menu-driven user interaction

## Concepts Used
- ArrayList
- HashMap
- Java Collections Framework
- Scanner class

## How to Run
1. Compile the program using:
javac OnlineShoppingCart.java

2. Run the program:
java OnlineShoppingCart
